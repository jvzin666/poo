POO - Definições
- Classe (concreta ou abstrata)
 é uma implementação de uma classe abstrata, permitindo a criação de instâncias.

- objeto
é um modelo ou blueprint que define os atributos (propriedades) e métodos (comportamentos) que todos os objetos dessa classe terão.

- Instancia
é um objeto específico que é derivado de uma classe. Cada instância possui seus próprios atributos e comportamentos definidos pela classe.

- atributo
 um molde ou blueprint para criar objetos. Os atributos são as características ou propriedades descritas na planta, como cor da parede, número de quartos, área em m². 

- constante
 é um paradigma que organiza o software em objetos que interagem entre si. 

- interface
 é um contrato que define um conjunto de métodos que uma classe deve implementar. 

- modificadores de acesso
são palavras-chave que controlam a acessibilidade de atributos e métodos.
